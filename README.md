# Eternal

**Legends Fade. Eternal Remains.**

Eternal is a Shadow Fight 2-inspired browser fighting game built with Next.js and TypeScript. Play as the Shadow — a silhouette warrior battling through a tournament of Sealers across seven arenas. Fight with punches, kicks, roundhouses, rolls, blocks, and rage-powered supers while a dynamic Chinese-inspired soundtrack intensifies with the action. Features local 2-player versus, mobile touch controls, WebGL post-processing, combo tracking, and an RL Ghost mode powered by reinforcement learning. Run `npm install` and `npm run dev` to play.
